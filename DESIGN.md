# SkillHire Design System

## 1. Core Colors (Shadcn/Inter)
- **Background**: White (#ffffff)
- **Foreground**: Deep Black (#09090b)
- **Primary**: Black (#09090b) for key actions
- **Primary Foreground**: White (#fafafa)
- **Muted**: Light Gray (#f4f4f5)
- **Muted Foreground**: Medium Gray (#71717a)
- **Border**: Pale Gray (#e4e4e7)
- **Card**: White (#ffffff) with subtle border

## 2. Typography
- **Font**: Inter (Sans-serif) / Geist Sans
- **Headings**: Bold, tight tracking
- **Body**: Clean, legible, 14px/16px

## 3. UI Components
- **Buttons**: Rounded (0.5rem), Primary is Black/White, Secondary is Ghost/Outline.
- **Cards**: Bordered, subtle shadow on hover, clean padding.
- **Inputs**: Simple bordered fields with clear placeholders.
- **Badges**: Pill-shaped, used for tags and status.

## 4. Layout
- **Container**: Centered, max-width 1200px (desktop).
- **Spacing**: Generous whitespace (gap-4, gap-6).
- **Grid**: Responsive grid for job cards (1 col mobile, 2 col tablet, 3 col desktop).

## 5. Specific Elements from Global CSS
- Radius: `0.625rem` (approx 10px)
- Animations: `tw-animate-css` included.

## 6. Design System Notes for Stitch Generation
**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Desktop-first
- Theme: Light, specific "Shadcn" aesthetic (clean, bordered, minimal)
- Background: White (#ffffff)
- Primary Accent: Black (#09090b) for buttons
- Text Primary: Black (#09090b)
- Text Secondary: Gray (#71717a)
- Cards: White surface, thin border (#e4e4e7), rounded-lg
- Font: Sans-serif (Inter style)
- **Iconography**: Lucide style (clean strokes)
